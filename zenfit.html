Zenfit privacy policy This privacy policy governs your use of the software application Zenfit (“Application”) for mobile devices that was created by Pete Cooke software. The Application is a simple puzzle game based rotating and flipping shapes into larger shapes. The application does not collect any information about the user and the only data held is the number of games played at each level. No data is collected from the user and no data is transmitted from or to the application. No data is shared with any 3rd parties.
